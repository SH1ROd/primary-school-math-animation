欢迎来到**方块数学大陆**！这里不是枯燥的考卷，而是一场关于红石逻辑、身法预测和史诗成就的思维挑战。

### 📂 文件结构说明

为了确保传送门（跳转链接）正常工作，请确保以下文件处于**同一个文件夹**内：

* `index.html` —— **【核心主控台】** 所有的任务从此开启。
* `01路程计算与基础相遇.html` —— 基础跑酷关卡。
* `02“路程和”的动态变化.html` —— 进阶追及挑战。
* `03折返跑问题.html` —— 终极小狼折返成就。

---

### 🕹️ 游玩指南 (How to Play)

1. **开启传送门**：用电脑或手机浏览器打开 `index.html`。
2. **接受任务**：点击任意一个卡片（如“史蒂夫的村庄跑酷”），进入对应的实验关卡。
3. **预测数值**：
* 在任务卡片的输入框里填入你推理出的数字（分钟或米）。
* 点击 **“运行预测/验证”**。


4. **动态推演**：你会看到角色（史蒂夫、僵尸或小狼）根据你的指令开始移动。
5. **解锁成就**：
* 如果预测准确，下方会弹出 **“🎯 完美结论”**，并同步解锁上方的 **“🔒 专家解析”**。
* 如果预测错误，系统会提示你差了多少方块，方便你修正红石电路（大脑逻辑）重新挑战！
