## 🕹️ 游玩说明

### 1. 开启实验室

在电脑或手机上直接点击 **`index.html`**。你会看到左右分割的酷炫界面，鼠标滑动或手指点击即可“扩张”你心仪的维度。

### 2. 任务指引

* **输入预测**：在任务卡片中填入你计算出的答案。
* **点击运行**：观察动画推演，看看你的红石电路（大脑）是否运行正确。
* **解锁解析**：只有预测完美准确，才会解锁隐藏的“专家解析”并获得勾选勋章。

### 3. 维度切换

如果你玩腻了标准版，随时可以退回到主页，跳入 MC 维度感受史蒂夫和僵尸的极限追逐。

## 🛠️ 技术小贴士 (Technical Tips)

* **浏览器推荐**：请使用 **Chrome、Safari 或 Edge** 浏览器以获得最丝滑的动态扩张效果。
* **移动端注意**：如果你在**微信**里收到这个文件，请点击右上角的 `...` 菜单，选择 **“用浏览器打开”**，否则页面之间的跳转可能会被微信拦截。
* **保持同步**：请勿随意更改文件名，否则传送门会因为找不到坐标而失效！
