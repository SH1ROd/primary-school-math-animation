## 🏗️ 逻辑实验室挑战题：消失的“时间”
[开始互动！](https://sh1rod.github.io/catch-up/)

### 【题目背景】

在美丽的青青草原上，有两条笔直的小路。小明和小红分别站在路的两端 A 点和 B 点，他们之间相距 **2000 米**。

* **小明**（穿着蓝色衣服）的速度是 **60 米/分钟**。
* **小红**（穿着粉色衣服）的速度是 **40 米/分钟**。
* 他们同时出发，面对面走来。
* 小明带了一只顽皮的**金毛犬**，它的速度很快，是 **80 米/分钟**。

### 【核心任务】

小明出发的一瞬间，小狗也跟着跑了出来。
这只小狗有个奇怪的习惯：它只要一碰到小红，就立刻掉头往小明跑；一碰到小明，又立刻掉头往小红跑。它就这样在两人之间不停地折返跳跃……

**请问：当小明和小红手拉手相遇的那一刻，这只顽皮的小狗一共跑了多少米？**
